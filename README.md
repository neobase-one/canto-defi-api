## Canto DeFi API

### Local Dev

1. Install `nvm`
```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
```
2. Install `node 16.0`
```sh
nvm install 16.0
```
3. Use `node 16.0`
```
nvm use 16.0
```
4. Install deps
```sh
yarn install
```
5. Run Server
```sh
yarn run redo
```
6. Server Running at [http://localhost:4000](http://localhost:4000)

### Docker Compose
```
docker compose up -d
```
